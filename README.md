# Web Scraping with Pandas 🌐⚽

A data collection project using **Python** and **Pandas** to scrape
and organize football data from the web.

## 📌 Project Overview
This project reads **football match data** directly from URLs
using Pandas — no browser needed! It covers multiple leagues
and multiple seasons (2015–2026).

## 📂 Files
| File | Description |
|------|-------------|
| `project_1_web_scraping_with_pandas.ipynb` | Main Jupyter Notebook |

## 📋 Project Sections
| Section | Description |
|---------|-------------|
| **1** | Read a single CSV from a URL |
| **2.1** | Read multiple leagues from URLs |
| **2.2** | Read multiple seasons (2015–2026) |
| **3** | Organize all data into a dictionary |

## 🏆 Leagues Covered
| League | Code |
|--------|------|
| English Premier League | E0 |
| English League 1 | E1 / E2 |
| English League 2 | E3 |
| English Conference | EC |
| Spanish La Liga | SP1 |
| Spanish Segunda Division | SP2 |
| German Bundesliga | D1 |

## 📅 Seasons Covered
- **2015–2026** (11 seasons)

## 🛠️ Libraries Used
- `pandas`

## 🌍 Data Source
- [football-data.co.uk](https://www.football-data.co.uk)

## ▶️ How to Run
1. Clone the repo
2. Open `project_1_web_scraping_with_pandas.ipynb`
   in Jupyter Notebook
3. Run all cells
4. Data is loaded directly from the web — no download needed!
